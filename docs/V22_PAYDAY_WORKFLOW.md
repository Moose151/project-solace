# Project Solace v22 — Payday and cycle workflow polish

This update adds current/next cycle viewing, cycle closeout, bill detail pages,
calendar bill links, a data health check, and a setting for how bills due on
payday should be assigned.

## Key behaviours

- Pay Cycle can switch between current and next cycle.
- Payday Checklist can switch between current and next cycle.
- Cycle Closeout gives a summary of paid, skipped, and still-unpaid bills.
- Bill detail pages show upcoming occurrences and recent history.
- Calendar bill events are tappable and open the bill detail page.
- Settings includes whether bills due on payday belong to the new or previous cycle.
- Health Check highlights common setup/data issues.
