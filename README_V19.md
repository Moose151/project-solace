# Project Solace v19 - Mobile-first polish

This update keeps the HomeStack warm theme and improves phone/tablet usability for household users.

Highlights:
- Mobile bottom navigation for the most-used routes.
- Larger touch targets for buttons, nav links, and form controls.
- Bills page has a mobile card-list view while preserving the desktop sortable table.
- Dashboard/action buttons stack cleanly on phones.
- Quick links and common action groups are easier to tap.
- Added docs/MOBILE_UI_NOTES.md for reusing these rules in Project Meridian and the future home dashboard.
